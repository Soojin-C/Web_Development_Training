A Study Space (2020)

This project was made for HACKNYU. It is a website that was targeted to students of NYU. It was built to help students find study partners and study spaces easily based on the subjects a user selects.

Features

  • Login and Logout for different users
  • Find spaces to study (NYU API)
  • Add tags to users profile to filter people
  • Find people in the database based on the tags mentioned above

Built with:

  • Python Backend
  • MySQLite Database
  • HTMl / CSS Frontend
  • BootStrap Front End Framework

Scribbl+ (2019)

This project was made for Software Development class project. It is a website that simulates the game of Scribble.io, a multiplayer drawing game.

Features

  • The drawing game playable with others
  • Message sending for guessing using web sockets
  • Color wheel, eraser and fill bucket to use for drawing

Built with:

  • Python Backend
  • HTMl/CSS/JavaScript Frontend
